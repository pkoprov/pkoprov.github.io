# Pavel Koprov — Personal Site

This repository contains the source for [pkoprov.github.io](https://pkoprov.github.io), the personal site of **Dr. Pavel Koprov**, Senior R&D Engineering Specialist focused on robotics, digital twins, and industrial AI. The site is built with the [Jekyll](https://jekyllrb.com/) static-site generator using the Minimal Mistakes theme and is deployed through GitHub Pages.

## Highlights
- Overview of Pavel's experience in industrial engineering and advanced manufacturing.
- Dedicated pages for publications, research projects, and an online CV.
- Blog-style posts for sharing updates in robotics, digital twins, and industrial AI.
- Built-in site search and responsive design courtesy of Minimal Mistakes.

## Repository Structure
- `_config.yml` — Site configuration including navigation, author profile, and metadata.
- `_pages/` — Static pages such as About, CV, Projects, and Publications.
- `_posts/` — Time-stamped updates and articles.
- `_data/` — Supplemental data files for navigation and author information.
- `assets/` — Images, documents, and other static assets.
- `index.html` — Landing page layout.

## Local Development
1. Install Ruby and Bundler (Ruby 3.1 or later recommended).
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Serve the site locally:
   ```bash
   bundle exec jekyll serve
   ```
4. Visit `http://localhost:4000` in your browser to preview changes.

## Deployment
Pushes to the `main` branch are built and deployed automatically by GitHub Pages. Ensure changes are committed and pushed to publish updates.
